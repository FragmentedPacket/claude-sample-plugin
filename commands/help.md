# Help Command

Displays help information about available commands and plugin capabilities.

## Usage

```
/help [command]
```

## Description

This command provides documentation and usage information for the plugin. When called without arguments, it lists all available commands. When called with a specific command name, it shows detailed help for that command.

## Arguments

- `command` (optional): The name of a specific command to get help for.

## Examples

```
/help
```
Lists all available commands.

```
/help hello
```
Shows detailed help for the hello command.

## Instructions

When this command is invoked:
1. If no argument provided, list all available commands with brief descriptions
2. If a command name is provided, show detailed usage for that command
3. Include examples where helpful

# Hello Command

A simple greeting command that demonstrates the basic structure of a slash command.

## Usage

```
/hello [name]
```

## Description

This command greets the user by name. If no name is provided, it uses a default greeting.

## Arguments

- `name` (optional): The name to greet. Defaults to "World" if not provided.

## Example

```
/hello Claude
```

Output: "Hello, Claude! Welcome to the sample plugin."

## Instructions

When this command is invoked:
1. Check if a name argument was provided
2. Generate a friendly greeting message
3. Include information about the plugin capabilities
